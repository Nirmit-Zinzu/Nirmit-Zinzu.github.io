Mark Book App



Language

- Java

Time

- July 2016

Description


Following application allows user to store the final grade of each course.
Moreover ...
  • User can add Course Name, Credit, Year, Grade and Season (Fall, Winter or Summer)
  • View all the course grades
  • Edit and Update previous entries
  • Delete specific or all entries
  • Set GPA (Grade Point Average) Scale
  • Calculates CGPA (Culminating Grade Point Average)
  • Calculates SGPA (Seasonal Grade Point Average)
  • Calculates final percent grade
  • Provides performance status
