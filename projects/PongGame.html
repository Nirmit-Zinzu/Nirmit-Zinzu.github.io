Pong Game



Language

- Java

Time

- Grade 12, May 2015

Description


This is a simple computer versus player pong game. User simply has to move the pallet using arrow keys. Score is shown at the bottom of the screen. Also, top and bottom of the screen act as a wall. Therefore, upon a hit, the ball will bounce accordingly. To win, user has to score ten points.
