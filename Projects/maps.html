MAP

LANGUAGE C++

TIME January - April, 2017

DESCRIPTION

Built a software program that provides functionality similar to Google Maps, plus some other features. The program incorporates following functionality:

  • Reads in a database of all the intersections and streets in a city.
  • Draws the resulting map and lets the user interact (pan, zoom, highlight, search for locations, etc.) with it.
  • Finds travel routes between two intersections in the city and gives directions to a user.
  • Finds a good order of deliveries and a good driving path for a courier company driver to complete his/her list of daily deliveries.
