Clothing Visual Search

LANGUAGE Python, Java

TIME Capstone Project, Septmeber 2019 - Present

DESCRIPTION

An android app which runs a neural network to return pictures of clothing items similar to a user-provided input.

  • The first goal: develop a deep learning framework that returns pictures of clothing items similar to a user-provided image. The general clothing categories that our model will consider include t-shirts, sweaters and hoodies, pants and jeans, jackets, shirts and blouses, dresses and skirts, and shorts.
  • The second goal: develop a mobile application to showcase our design by simulating an arbitrary fashion retailer.
