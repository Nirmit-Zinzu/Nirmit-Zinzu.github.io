SPYFALL

LANGUAGE Java

TIME August, 2016

DESCRIPTION

This is a modified version of the actual board game.

User can chose to enter their own location or chose a random one from the list. If everyone wants to play the game, it is wise to allow the app to chose a random location. Otherwise, a narrator (host of the game) can chose to enter a custom location.

After this, each person gets to see the location. User has to pass down the device and allow each members to see it. Among this, one of the players is a spy. The task of the game is to figure out which person is "Spy".
